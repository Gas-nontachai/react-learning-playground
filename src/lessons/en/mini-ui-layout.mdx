Designing the main layout early clarifies how pages, navigation, and reusable sections connect.

<br />

## Steps
1. Sketch the layout on paper or Figma (header, sidebar, content area)
2. Implement a responsive grid using CSS modules or Tailwind
3. Extract layout primitives (`AppShell`, `SidebarLink`, `PageHeader`)

<br />

## Example shell

```tsx
// src/components/layout/AppShell.tsx
import Link from 'next/link';

const links = [
  {href: '/', label: 'Dashboard'},
  {href: '/lessons', label: 'Lessons'},
  {href: '/students', label: 'Students'}
];

export function AppShell({children}: {children: React.ReactNode}) {
  return (
    <div className="grid min-h-screen gap-6 bg-slate-50 p-6 md:grid-cols-[220px_1fr]">
      <aside className="space-y-2 rounded-2xl border border-slate-200 bg-white p-4">
        {links.map((link) => (
          <Link key={link.href} href={link.href} className="block rounded px-3 py-2 hover:bg-slate-100">
            {link.label}
          </Link>
        ))}
      </aside>
      <main className="rounded-2xl border border-slate-200 bg-white p-6 shadow">
        {children}
      </main>
    </div>
  );
}
```

Use the shell in `app/layout.tsx` so every page inherits the structure.

<br />
