เป้าหมายคือให้เข้าใจ JavaScript ระดับที่สามารถอ่านโค้ด React ได้อย่างไม่ติดขัด

## หัวข้อ JavaScript ที่ต้องใช้ในการเรียน React

### 1. let และ const
ใช้ประกาศตัวแปร  
- let ใช้เมื่อค่าถูกเปลี่ยน  
- const ใช้เมื่อค่าคงที่

ตัวอย่าง:

```js
let count = 0;
count = count + 1;

const name = "React";
```

<br />

### 2. Arrow Function
รูปแบบการเขียนฟังก์ชันแบบสั้น กระชับ และใช้บ่อยมากใน React

```js
const add = (a, b) => a + b;
```

<br />

### 3. Object และ Array
พื้นฐานข้อมูลที่ React ใช้จัดการตลอดเวลา

```js
const user = { name: "Mint", age: 20 };
const numbers = [1, 2, 3];
```

<br />

### 4. Array Methods ที่จำเป็น
ใช้บ่อยที่สุดเวลาแสดงรายการข้อมูลใน React

- map  
- filter  
- find  
- some / every  

ตัวอย่าง:

```js
const numbersList = [1, 2, 3];
const doubled = numbersList.map(n => n * 2);
```

<br />

### 5. Destructuring
ใช้แยกค่าจาก array หรือ object  
เป็น syntax ที่พบใน React ทุกวัน

```js
const userInfo = { name: "Mint", age: 20 };
const { name: userName, age: userAge } = userInfo;
```

<br />

### 6. Spread Syntax
ใช้กระจายค่าของ array หรือ object  
สำคัญมากเวลาต้องอัปเดต state ใน React

```js
const nums = [1, 2, 3];
const moreNums = [...nums, 4];
```
