# 1.2 JSX

JSX คือรูปแบบการเขียนโค้ดที่ทำให้เราสามารถเขียนโครงสร้างลักษณะคล้าย HTML ภายใน JavaScript ได้โดยตรง  
React ใช้ JSX เพื่อทำให้การสร้าง UI อ่านง่าย เป็นธรรมชาติ และใกล้เคียงกับสิ่งที่เรามองเห็นบนหน้าเว็บ

<br />

## JSX คืออะไร

JSX ไม่ใช่ HTML แต่เป็น syntax ที่ถูกแปลงเป็น JavaScript อีกทีหนึ่ง  

ตัวอย่าง JSX:
```jsx
<h1>Hello JSX</h1>
```

เบื้องหลัง JSX จะถูกแปลงเป็นรูปแบบ JavaScript เช่น:
React.createElement("h1", null, "Hello JSX");

<br />

## ข้อควรรู้เกี่ยวกับ JSX

### 1. JSX ต้องมี element หลักห่อทั้งหมด
ผิด:
```jsx
<h1>A</h1>
<p>B</p>
```

ถูก:
```jsx
<>
  <h1>A</h1>
  <p>B</p>
</>
```

<br />

### 2. attribute ของ JSX ใช้แบบ camelCase
```jsx
<div className="box"></div>
<button onClick={handleClick}></button>
```

<br />

### 3. JSX สามารถแทรก JavaScript ได้
```jsx
const name = "React";
<h1>Hello {name}</h1>
```
<br />

### 4. JSX สามารถใช้เงื่อนไขได้
```jsx
{isLoggedIn ? <p>Welcome</p> : <p>Please log in</p>}
```
<br />

## ตัวอย่าง Component ที่ใช้ JSX
```jsx
function Welcome() {
  const message = "ยินดีต้อนรับสู่ React";

  return (
    <div>
      <h1>{message}</h1>
      <p>นี่คือ JSX ตัวอย่างพื้นฐาน</p>
    </div>
  );
}
```

<br />
