# 1.1 React คืออะไร

React คือไลบรารี JavaScript สำหรับสร้างส่วนติดต่อผู้ใช้ (User Interface)  
จุดเด่นของ React คือช่วยให้เราจัดการหน้าจอที่ซับซ้อนได้อย่างเป็นระบบ และแยกโค้ดออกเป็นชิ้นเล็ก ๆ ที่ดูแลง่ายขึ้น

React ถูกสร้างโดยทีมของ Facebook เพื่อแก้ปัญหาหน้าเว็บที่มีการอัปเดตบ่อยครั้ง และมีสถานะ (state) ที่เปลี่ยนไปตามการใช้งานของผู้ใช้

<br />

## React แก้ปัญหาอะไร

ก่อนมี React การจัดการ UI ด้วย JavaScript แบบเดิมมักมีปัญหา เช่น:

- โค้ดกระจัดกระจาย แก้จุดหนึ่งแล้วจุดอื่นพัง
- DOM อัปเดตเองหลายที่ ทำให้สับสน
- การกำหนดเงื่อนไขการแสดงผลซ้ำไปซ้ำมา
- ยิ่งหน้าเว็บใหญ่ การดูแลยิ่งยากขึ้นมาก

React เสนอวิธีคิดใหม่:  
ให้เราคิดว่า “UI = ผลลัพธ์ของ State”  
เมื่อ state เปลี่ยน UI จะอัปเดตเองแบบอัตโนมัติ

<br />

## แนวคิดหลักของ React

### 1. Component
React แบ่ง UI ออกเป็นชิ้นส่วนเล็ก ๆ เรียกว่า Component  
ทำให้สามารถนำกลับมาใช้ซ้ำ และแยกดูแลทีละส่วนได้ง่ายขึ้น

### 2. การจัดการ State
State คือข้อมูลที่เปลี่ยนได้ เช่น ข้อความในช่อง input, จำนวนสินค้าในตะกร้า  
React จะอัปเดต UI ให้สอดคล้องกับ state ที่เปลี่ยนไปโดยอัตโนมัติ

### 3. Virtual DOM
React ใช้ Virtual DOM เพื่อเปรียบเทียบความเปลี่ยนแปลง  
แล้วอัปเดตเฉพาะส่วนที่จำเป็น ช่วยให้เว็บทำงานลื่นขึ้น
