## แนวคิดสำคัญ
1. เข้าใจว่าบทนี้เป็นส่วนหนึ่งของ Section 0 – Fundamentals  
   และเป็นรากฐานก่อนจะไปเขียน React จริง  
2. ผูกคำศัพท์ใหม่ เช่น Browser / Server / Request / Response  
   เข้ากับสิ่งที่คุณเคยใช้ เช่น การเปิด YouTube หรือเว็บไซต์ข่าว  
3. อ่านตัวอย่างโค้ดแล้วลองอธิบายด้วยคำพูดของตัวเอง เพื่อทดสอบความเข้าใจ

<br />

## Web ทำงานอย่างไร (อธิบายให้เห็นภาพ)

1. เปิด Chrome หรือ Safari นี่คือ Browser  
2. พิมพ์ชื่อเว็บไซต์ เช่น google.com → Browser ส่งคำขอ (Request)  
3. Server ของเว็บไซต์รับคำขอ → ประมวลผล → ส่งข้อมูลกลับ (Response)  
4. Browser นำข้อมูลมาวาดเป็น UI เพื่อให้ผู้ใช้เห็นเป็นหน้าเว็บ

โครงสร้างแบบย่อ:
Browser → Request → Server  
Server → Response → Browser → Render UI
